# spacemacs
## Install
* Install the [spacemacs](http://spacemacs.org)
```
git clone https://github.com/syl20bnr/spacemacs ~/.emacs.d
```
* Run
```
install.sh
```
## Notes
### `ggtags` key bindings
  * `M-.` find by context
  * `M-*` move to the previous point on the stack
  * `M-]` find reference
  * `C-c M-g` grep
  * `C-c M-f` find file
  * `C-M-.` find regexp tag
